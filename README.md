# Pomodoro Timer

A Python desktop productivity app built with Tkinter to manage work cycles and breaks.

## ✨ Features
* **Automated Cycles:** Handles 25-minute work intervals, 5-minute short breaks, and a 20-minute long break after 4 rounds.
* **Visual Tracker:** Displays color-coded session states (Work/Break) and tracks completed cycles with checkmarks (`✓`).
* **Session Control:** Includes responsive `START` and `RESET` controls.
